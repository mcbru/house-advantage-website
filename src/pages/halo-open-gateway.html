HALo Open Gateway

A Simple API to Support Integration with Third Party Systems

HALo Open Gateway provides third-party systems that do not natively support S2S messaging with an easy to implement point-to-point integration solution to communicate with HALo CORE. A simplified API provides third-party systems with bidirectional access to player information and the transactional data necessary to bring these systems into the customer loyalty experience, and extend them to web portals, mobile clients, and other systems.
