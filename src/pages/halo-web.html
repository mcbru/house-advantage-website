HALo Web

Add Web Portals to Your Loyalty Solution

Leverage your existing web presence by deploying secure player portals to your existing web sites. HALo Web allows real-time access to player data and extends the reach of loyalty programs to wherever and whenever the customer wishes. HALo Web allows operators to integrate web portals in HALO CORE or their casino management system, enabling customers to access information about their loyalty program, profile, tier, balances and offers.

Access to this data is secured through the same security credentials loyalty members use to access the casino management system: Player ID and PIN.
